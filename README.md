# Readme  
Simply add this custom setting to the DefaultKeyBindings.dict file :   
  
MacBook-Pro-mac:~ mac$ cat ~/Library/KeyBindings/DefaultKeyBinding.dict  
{  
"~а" = (insertText:, "ә");  
"~г" = (insertText:, "һ");  
"~ж" = (insertText:, "җ");  
"~н" = (insertText:, "ң");  
"~о" = (insertText:, "ө");  
"~у" = (insertText:, "ү");  
}
